# E-Commerce Sales Dashboard Analysis

## Project Overview
This project focuses on analyzing e-commerce sales data to gain insights into business performance, customer behavior, and product trends. The analysis was performed using SQL for data extraction and Power BI for building interactive dashboards.

The goal of this project is to transform raw sales data into meaningful insights that help businesses make data-driven decisions and identify growth opportunities.

---

## Problem Statement
E-commerce businesses generate large volumes of sales data. However, without proper analysis and visualization, it becomes difficult to understand trends, customer behavior, and product performance.

This project aims to analyze e-commerce sales data and build interactive dashboards to track important business metrics such as profit, order volume, product performance, and sales trends.

---

## Objectives
- Analyze e-commerce sales data to identify key trends
- Track important business KPIs
- Understand customer and product performance
- Build interactive dashboards for business insights

---

## Tools and Technologies
- **SQL** – Data querying and analysis  
- **Power BI** – Data visualization and dashboard creation  
- **Excel / CSV Dataset** – Source dataset for analysis  

---

## Data Analysis Process

### 1. Data Collection
The dataset contains information about:
- Orders
- Products
- Sales revenue
- Profit
- Customer details

### 2. Data Analysis using SQL
SQL queries were used to:
- Filter and aggregate sales data
- Calculate key metrics such as profit and order volume
- Identify top-performing products and categories

### 3. Data Visualization using Power BI
Power BI was used to create interactive dashboards and visualizations including:
- Sales trends over time
- Product performance
- Profit analysis
- Customer insights

---

## Dashboard Features
The Power BI dashboard provides insights into:

- Total Sales
- Total Profit
- Order Volume
- Product Category Performance
- Customer Demographics
- Sales Trends

Interactive filters and visuals allow users to explore the data easily.

---

## Key Insights
- Identified top-selling products and categories
- Analyzed profit trends across different product segments
- Observed patterns in order volume and sales growth
- Provided insights to support business decision-making
